# StockWise

This project is created in a procedural way.

## About

StockWise is a PHP-based product management and mini e-commerce application. It allows users to browse products, filter by categories, manage stock, and add items to a simple shopping cart. The app includes secure user authentication and CRUD functionality, demonstrating skills in backend logic, session handling, and application structure.

## Launch the project

```bash
php -S localhost:8000
```
